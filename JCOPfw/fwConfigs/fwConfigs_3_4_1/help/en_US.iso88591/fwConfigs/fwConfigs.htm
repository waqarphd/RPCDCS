Component Name
 fwConfigs
Description

Provides access to read and write to the supported PVSS configs. Library functions and user interface panels are provided to manage the configs from both CTRL Script and User Interface.

The supported configs are:

Alert Handling (_alert_hdl)
Archiving (_archive)
Command/Message Conversion (_cmd_conv & _msg_conv)
DP Function (_dp_fct)
Original Value (_original)
Peripheral Address (_distrib & _address)
PV Range (_pv_range)
Smoothing (_smooth)
And in addition the Format and Unit of the common config can be set (_common)

Panel Name

fwDipEditDpeMapping.pnl
Introduction
 This panel is used while creating a complex subscription. It is used to manually select the dpe that will be subscribed to specified DIP tag.
Instructions The selection box contains all the possible (type compatible and not already used for subscription) dpes of given DP.

 

 

 

Dollar Parameters
Name Description  
$sText A caption that is being displayed above the selection box Required
$dsFieldName A dynstring with all the dpes to be displayed in the selection box converted to a string. Required

Component Name
 fwDIP
Description

Provides communication between PVSS and DIP. DIP is the Data Interchange Protocol which is used for exchanging summarised data between various systems at CERN

PVSS data points can be subscribed to DIP data or PVSS data points can publish their data over DIP.

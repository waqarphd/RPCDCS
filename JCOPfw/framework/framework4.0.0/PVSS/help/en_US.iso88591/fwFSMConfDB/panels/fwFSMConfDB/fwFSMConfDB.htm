Introduction

This is the FSMConfDB main panel. The panel allows you to add, change or delete configurators for a certain control unit or for a full tree.

Instructions
Select a tree or a sub-tree to see the list of all the control units belonging to it.
You can manage each configurator individually by selecting its correspondent control unit and right click to see the cascade options. You can also create all configurators at once by selecting the tree and clicking on Add ALL configurators.
 
Add/Change a configurator:
This option allows you to add or change the parameters of a specific configurator. Selecting the option will bring you to the addConfigurator panel. Follow the instruction given in the help for this panel.
 
Delete a configurator:
Select the control unit of the configurator you want to delete from the list and right click on it. A popup window will ask you to confirm this operation.
 
User Recipes:
This option allows the user to add specific recipes not linked to commands in the list of recipes to be handled by the configurator. Selecting the option will bring you to the userRecipes panel.
 
Add all configurators:
Depending on the mode of operation that you want to use, select the check button "apply recipes". In that mode you can choose what is the timeout for all your configurators. This option will create all the configurators with the same parameters: apply recipe and timeout, if you want to use different parameters for your configurators use the add configurator option.
Advanced settings
Button to open the Advanced settings panel. In this panel you can modify the behavior of the configurator when recipes are loaded.

Introduction

Change password panel

The standard PVSS change password panel, customized for use with the JCOP Framework. Allows to change own password.

Figure 1: Change password panel
Instructions
To change the password:
  • Type in your current password in the "Current Password" field and press ENTER
  • Type in your new password in the "New Password" field and press ENTER
  • Type in your new password again in the "Verify New Password" field and press ENTER

If the current password is incorrect, you will be informed by an error message.

In the Integrated Access Control mode (i.e. using the Access Control Server), the new password will be sent to the server then synchronised on all managed systems.

To cancel:
Press the Cancel button
To change the password of another user:
Use the Edit User panel. Note that user administration access rights are needed; in the integrated mode the change may only be performed on the Access Control Server.
To change the logo (pitcture):
Copy the file (in the BMP format) containing the logo to pictures/logo.bmp . This will alter the logo used by the password change panel and the Login panel.
Restrictions
  • Empty passwords are not allowed due to security reasons.

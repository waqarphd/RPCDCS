Component Name fwConfigurationDB
Description

Configuration Database Tool

The ConfigurationDB tool is used to store versions of static and dynamic configuration data for PVSS (Framework and non-Framework) devices in the Oracle database, and to configure the systems (apply settings) using the configuration data stored there.

A "getting-started" guide for the use of the fwConfigurationDB libraries is available at the main library help page.

Dependencies
  • fwCore
Additional software required
  • Oracle client
  • CtrlRDBAccess >= 0.45

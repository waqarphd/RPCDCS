Panel Name

fwWienerTripsOverview.pnl
Introduction

This panel is intended to be used as a reference panel for showing the trip settings of a Wiener device.
Instructions

When the panel loads, it displays the information relating to the Wiener device specified $sDpName.
Restrictions

None

 

 

Dollar Parameters
Name Description  
$sDpName

The name of the Wiener data point to act on. Must be of type FwWienerPS, FwWienerFanTray or FwWienerCrate

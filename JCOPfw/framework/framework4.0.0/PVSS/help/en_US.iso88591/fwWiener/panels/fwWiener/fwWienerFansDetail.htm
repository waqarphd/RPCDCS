Panel Name

fwWienerFansDetail.pnl
Introduction

This panel is used to show the detailed information about the fans of Wiener devices, such as individual speeds, average speed and nominal speed.
Instructions

This panel shows all the data about the fans. A button is available to change to fan speed setting.
Restrictions

None

 

 

Dollar Parameters
Name Description  
$sDpName

The name of the data point to show details of. This data point must be of type FwWienerCrate, FwWienerPS or FwWienerFanTray.

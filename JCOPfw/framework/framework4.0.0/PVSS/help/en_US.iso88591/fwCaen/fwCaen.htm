Component Name
 fwCaen
Description

This component contains the Framework devices required to work with the CAEN hardware. The access to the devices is through the CAEN OPC server. Currently, the following hardware is supported:

  • SY1527 (and SY2527)
    • Boards: A1517A, A1511, A1511B, A1512, A1513A/B,  A1516B, A1518A/B, A1519, A1519B,  A1520P/PE, A1526P/N, A1534P/N, A1535P/N, A1732P/N, A1733P/N, A1734P/N, A1735P/N, A1738P/N, A1821P/N, A1832, A1833P/N, A1833BP/N, A1932AN
    • Special: A1676 (Easy branch controller), A876 (CMS MDT controller)
    • Easy boards:
      • LV: A3006, A3009, A3009B, A3016, A3025, A3025B, A3025D, A3050, A3050D, A3100
      • HV: A3501P/N, A3506P/N, A3507P/N, A3512P/N, A3535P/N, A3540P/N
      • Power converter: A3486, A3486S, A3485, A3485A
      • Special: A4601H/F, A4602, A4603, A3801, A3801A, A1396
The old crate models are also supported:
  • SY127
    • Boards: A230, A326P/N, A327P/N, A329P/N, A332P/N, A333P/N, A334P/N,  A335P/N,  A431P/N, A432P/N,  A433P/N, A434P/N, A634P/N
  • SY403
    • Boards: A503P/N, A504P/N
  • SY527
    • Boards: A373A, A516, A732P/N, A733AP/AN, A734P/N, A753P/N, A821P/N, A832P/N, A833A/AP/AN, A834P/N, A835P/N
The power supplies of the SY1527 family are accesed via Ethernet. The old models are accessed via CAENET.

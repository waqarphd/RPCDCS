Panel Name

fwDeviceEditorNavigatorClipboard.pnl
Introduction
When a device is cut from the Logical view, it is placed in the Clipboard. This panel is a brower and selector for the Clipboard, when pasting back to the Logical tree.
Instructions
The panel displays the devices that have been cut from the Logical view organized in a tree. It is possible to select any subtree by choosing one of the devices. When pressing the 'Ok' button, it will be checked if the selected device is of the right type to be pasted back in the Logical tree position from where the Clipboard was open.

It is possible to remove a subtree from the clipboard by right clicking on it and then choosing 'Remove'.

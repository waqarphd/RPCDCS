Panel Name

fwMagnetSubscription.pnl
Introduction
 This is the main panel of the fwMCS component. It subscribes publications availables for a selected experiment and generates the necessary data in PVSS.
Instructions

The header of the panel contains two selection boxes:

  • One for selecting the DIP API manager config DP
  • And another one for selecting the experiment.

After selecting these options the table with the available DIP publications is filled, and also the publication content panel with the details for the first publication showed in the table.

It is possible to subscribe a publication if one of them from the table is selected and the button "Add Subscription" is clicked.

If it is the first time that is made a subscription for an experiment, a Device Unit Type is created for the FSM. The states of this DU Type are: ON, OFF, RAMPING_UP, RAMPING_DOWN and ERROR.

It is also possible to delete a subscription. It is necessary to select a subscription from the table with the DIP Subscriptions and then click over the button "Delete Subscription".
Restrictions
  • To see the publications and make a subscription, the DIP manager has to be running.
  • To create the DU Type is necessary that the publication contain a tag for the current value and another for the ramping.

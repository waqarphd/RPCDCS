Panel Name

fwAlarmSummaryOperationPopupPanel.pnl
Introduction

This panel is used to view the state of a summary alarm on a data point element. The current state of the alarm is indicated by a coloured box.

The user can also mask (deactivate) and unmask (activate) as appropriate.
Instructions

When the panel loads, it displays the current state of the alert and the name of the data point element which is being viewed.

Two buttons are available to perform actions. These buttons are disabled when the action is not relevant to the current state of the alarm. The buttons are:

Mask - Mask the alarm. The alert config is deactivated. The configuration is kept but no alarm is generated.

Unmask - Unmask the alarm. The alert config is activated.
Restrictions

This panel can be used to only to view summary alarms. For viewing analog and digital style alarms, use fwAlarmOperationPopupPanel.pnl.

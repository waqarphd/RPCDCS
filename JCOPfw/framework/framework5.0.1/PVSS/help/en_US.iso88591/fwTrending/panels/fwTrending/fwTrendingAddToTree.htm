Panel Name

fwTrendingAddToTree.pnl
Introduction

This panel is used to add items to the Trend Tree. These items can be nodes to organise subelements and new or existing pages and plots..
Instructions

Depending on how you load the panel, you will either be presented with options to:

1) Create a new tree node

In this mode, you should enter a name for the tree node in the "Node Name" field. This name is used in the tree to represent this node. The name must meet the requirements for valid data point names.

2) Create a new plot/page in the tree

In this mode, you must first select whether you want to create a Trending Plot or a Trending Page. Next, you should enter a data point for this new item in the "DP Name" field. Finally, you should enter a name in the "Name in tree" field. This name is used to represent this new item in the Trending Tree. Both the data point name and name for the tree must meet the requirements for valid data point names.

3) Add an existing plot/page in the tree

In this mode, you must first select whether you want to add a Trending Plot or a Trending Page. Next, in the list below, you will see all the items of the selected type that are available to be added in the tree. Select the desired plot/page from this list. Finally, you should enter a name in the "Name in tree" field. This name is used to represent this existing item at this location in the Trending Tree. The name for use in the tree must meet the requirements for valid data point names.

Once you have entered the necessary data, press OK to complete the action or press Cancel to close the panel without modifying the Trend Tree.
Restrictions This panel just prepares the data to add the item in the tree. It does not actually carry out the action. That is done by the panel which calls this panel.

 

 

 

 

 

Dollar Parameters
Name Description  
$2 The name of the tree item below which the new item should be created (used for display only). Required
$3 A comma seperated list of data point types which can be added in the tree at this location. Required
$4 A comma seperated list of data point elements from which the proposed "Name in tree" should be read. For each entry in $3 there should be value in $4 saying which dpe to read for that dp type. Each entry should be in the following format "DP" followed by the data point element from which to read. In the case of the Trending Plot, the value reads "DP.plotTitle". Required
$mode

Controls the mode in which the panel runs. Should be one of the following values:
"addnode", "addnew" or "addexisting" (not case sensitive)

Introduction

Advanced Settings panel:

Instructions
These settings were included for very specific cases. Please ignore these settings and keep the default values unless you are one of those special cases.
- recipe pattern
A recipe pattern can be set to filter the list of recipes to be loaded by the configurator.
Only recipes containing the recipe pattern will be loaded.
- ignore device list
if active, the configurator ignores the list of devices and loads the full recipe. With this option enabled, the stardard fwFSMConfDB functions cannot be used. The user must create his/her own functions to apply the recipes.

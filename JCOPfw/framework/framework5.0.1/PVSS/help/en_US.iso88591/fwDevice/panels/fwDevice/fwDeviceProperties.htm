Panel Name

fwDeviceProperties.pnl
Introduction
 This panel provides an easy way to have an overview of the configuration of all the elements of the Framework device that is passed as parameter, as well as the possibility to change the that configuration.
Instructions When the panel is opened a table is displayed where each row corresponds to one of the elements of the device.  The first two columns display the property name and PVSS datapoint element name. The additional columns correspond to the PVSS configs and indicate whether the config has been set for each particular element. Clicking on a cell will take you to the panel to configure the config (given by the column) for the selected element (given by the row).

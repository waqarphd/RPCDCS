Component Name
 fwWiener
Description

This component contains the Framework devices required to work with the Wiener hardware. The access to the devices is through one of the Wiener OPC servers.

Currently through the Wiener KVASER OPC server you can control CAN-based Wiener Fan Trays, Crates and Power Supplies.

With the Wiener OPC Server you can access the new Ethernet Wiener MARATON hardware.
Dependencies
 fwCore
Additional software required

Krakow Wiener KVASER OPC server available at http://itcofe.web.cern.ch/itcofe/Services/OPC/RecommendedTools/Servers/WIENER/welcome.html

and/or

Wiener OPC Server for Ethernet MARATON hardware.

Panel Name

fwWienerTempareturesDetail.pnl
Introduction

This panel is used to show the detailed information about the temperatures of Wiener devices, such as the state of each temperature probe as well as tables giving detailed readings and settings of warning and temperature limits
Instructions

The panel shows the current state through a series of LEDs and a table of values. There is also a table to change the settings relating to temperature warning and limit thresholds. These values can only be modified when the device is switched off. Then, the cells in the "Setting" column will become white which indicates the setting can be modified. To modify the setting, simply click in the white cell in the "Setting" column corresponding to the parameter you want to set.
Restrictions

None

 

 

Dollar Parameters
Name Description  
$sDpName

The name of the data point to show details of. This data point must be of type FwWienerCrate, FwWienerPS or FwWienerFanTray.

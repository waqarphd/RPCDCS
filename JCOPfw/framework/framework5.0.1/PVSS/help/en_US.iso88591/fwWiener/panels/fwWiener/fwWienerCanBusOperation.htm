Panel Name

fwWienerCanBusOperation.pnl
Introduction

This panel is used to monitor and operate the Wiener CAN bus framework devices.
Instructions

The operation panel is divided into several sections. Along the top is the general information like data point name, description and the baud rate of the CAN bus. There is also the possibility to go to the framework "Action" panel to peform actions on alerts and an indication of the state (using the framework colour convention) of the summary alert of the device.

Below this are three sections that give an overview of the state of the bus, including error states, buffer overflows etc. In general when the LEDs show the standard framework OK colour, then the corresponding property is in a good state.

At the bottom left of the panel are a set of push buttons to perform actions on the CAN bus device. Actions are available to turn the CAN bus ON/OFF, as well as resetting and flushing queues.

Finally, the bottom right shows the current value of some important CAN bus statistics. The values will update live. By clicking in the "Details" column at the end of the row, a detailed display of the attributes of the corresponding value is shown.
Restrictions

None

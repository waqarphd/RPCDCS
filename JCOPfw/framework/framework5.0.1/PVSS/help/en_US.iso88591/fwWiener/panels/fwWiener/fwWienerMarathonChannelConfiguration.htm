Panel Name

fwWienerMarathonChannelConfiguration.pnl
Introduction

This panel is used to set the configuration of the Wiener Marathon Channel framework devices.
Instructions

When the panel loads, it displays the current configuration of the Wiener Marathon Channel device. The device data point name and the position of the channel in the parent device are also shown.

The description of the device can be modified by entering the new description in the "Description" field and then pressing the enter key to save the new value.

Tabs are provided for switching between various different types of configuration. Tabs are available for setting the following configurations:

  • Peripheral addressing
  • Archiving (and smoothing on the archived data)
  • Alarms on device properties and a summary alarm for the device
  • PV Range limits
  • Conversion settings (input and output)
  • Smoothing
  • Units
  • Format

The standard framework panels are used for the configuration of these settings.
Restrictions

None

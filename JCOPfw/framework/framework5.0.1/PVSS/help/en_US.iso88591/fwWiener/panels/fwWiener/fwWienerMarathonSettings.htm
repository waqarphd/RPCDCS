Panel Name

fwWienerMarathonSettings.pnl
Introduction

This panel is used to set certain settings of the Wiener Marathon devices. The settings entered here are included in the OPC server XML config file if this is exported using the fwWienerDriverConfiguration panel.
Instructions

When the panel loads, it displays the current values stored in PVSS for the given parameters. In the case of the Wiener Marathon devices, this is just the IP number of the device.

New settings can be entered as desired. To save the settings, press the Apply button. To close the panel without changing the settings, press the Close button.
Restrictions

The values set in the panel are not automatically applied to the Wiener Marathon device. The panel just saves the settings to PVSS. To actually change to IP address of the device, there is a seperate method to configure this directly on the hardware. It is important that the value in PVSS matches the real IP address if you plan to export the OPC server config file using the fwWienerDriverConfiguration panel.

Panel Name

fwWienerChannelOperation.pnl
Introduction

This panel is used to monitor and operate the Wiener channel framework devices.
Instructions

The operation panel is divided into several sections. Along the top is the general information like data point name, description and the position of the Wiener channel in its parent device. There is also the possibility to go to the framework "Action" panel to peform actions on alerts and an indication of the state (using the framework colour convention) of the summary alert of the device.

Below this are two sections that give an overview of the state of the channel, including error flags, current and voltage values. When the LEDs show the standard framework OK colour, then the corresponding property is in a good state.

At the bottom of the panel is a table which gives detailed information on all of the settings available for the Wiener channel. The table provides the current setting, a readback of the setting from the hardware and the maximum and minimum allowed limits for each parameter.

For a Wiener channel that belongs to a Wiener Crate, these settings can not be modified. For a Wiener channel that belongs to a Wiener Power Supply, the settings can be modified. When the Power Supply is on, only the voltage and current can be set. When the Power Supply is off, all the settings can be modified. It is possibleto tell which settings can be changed at any given time because the cell of the table in the "Setting" column will become white. To change a setting simply click in the white cell in the "Setting" column.
Restrictions

None

Importing subscriptions or publications

 

To import, start the DEN and push the "Go to Editor" button. Subsequently, push the External-Application sub-component you want to import. You will get the following two panels on your screen.

 

     

 

Now press the navivator to select the configuration filename. In the example below we only have one file for the Accelerator-Data, but for most sub-components, we have a file per LHC experiment.

 

 

Select a file to import and the contents of the file will be loaded in the window of the configuration panel. Now one can select all or only portions to be imported in the WINCC-OA environemnt. To help your decision making, one can also display the description of items in the lower viewing window by simply clicking on the corresponding DIP item (see below).

 

     

 

Finally, one only needs to press the "Import Configuration" button to import the selected subscriptions or publications.

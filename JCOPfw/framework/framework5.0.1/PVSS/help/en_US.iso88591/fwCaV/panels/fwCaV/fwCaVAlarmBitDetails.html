Panel Name

fwCaVAlarmBitDetails.pnl
Introduction
 Panel for providing detailed information about alarm/warning bit
Instructions

Simple panel for displaying information that is being passed as $-parameters.

It is possible to change the text of the bit description
Restrictions If the Access Control is installed and configured and the user is not an Expert, the Change button will be disabled.

Dollar Parameters
Name Description  
$iBit Datapoint of the plant/loop  
$sDesc Description of the given alarm/warning bit  
$sTitle Title of the panel - "Detailed bit information" on above picture  
$sVar Variable name - "Tile_outlet_temp_flt" on above picture  
$sWord Word name - ".Warnings.warningWord01"

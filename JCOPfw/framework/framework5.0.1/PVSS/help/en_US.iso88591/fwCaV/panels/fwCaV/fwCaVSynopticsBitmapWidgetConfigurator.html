fwCaVSynopticsBitmapWidgetConfigurator.pnl
Introduction
 Panel for managing the Synoptics Bitmap Widget Types
Instructions

This panel is intended for the creators of the Widgets that are going to be used along with Synoptics Bitmap facility. The widgets must follow a certain conventions described here.

When one creates such widget, it is necessary to register it - provide path to the reference panel, provide description of the parameters and additional parameters.

Panel Name

fwCaVSynopticsBitmapNewPanel.pnl
Introduction
 Panel for configuring the Synoptics Bitmap instance
Instructions

This panels allows to create new and modify already existing Synoptics Bitmap instance.

User can select the bitmap image and specify the desired size of the bitmap in pixels. It is also possible to select the size that the plot will have.
Restrictions  

Dollar Parameters
Name Description  
$bNew Mode of operation: 0 - modify; 1 - create new  
$sBitmapName In case of modify - which Bitmap Instance to modify

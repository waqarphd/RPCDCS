Panel Name

fwTrendingSaveAsSelector.pnl
Introduction

This panel is used to select a data point to save the plot or page configuration to.
Instructions

When you select "Save As..." this dialog will appear, usually with a proposed name for the data point name.

In the text field you can enter any valid data point to which the configuration will be saved. If this is an existing data point you will be warned that it will overwrite the previous configuration. If it is a data point which does not exist, this data point will be created for you.

Once you have entered the correct data point name, press OK to save the configuration. Or press "Cancel" to close the panel without saving the configuration.
Restrictions None

 

 

 

Dollar Parameters
Name Description  
$sDpType

The type of the data point required for the configuration. ie. FwTrendingPage or FwTrendingPlot

 Required
$sProposedName The proposed name for the new data point. If the value is left blank or not set, then no suggested name is given when the panel loads. Optional

Component Name
 fwTrending
Description

The Trending Tool provides a way of configuring predefined plots and pages which can be viewed at any time to give trending of the data in your system. Many features are supported which are not provided by the default PVSS widget or the PVSS VarTrend tool.

A Trending Page is a collection of Trending Plots. It is possible to add up to 6 plots on a page (2 columns and 3 rows).

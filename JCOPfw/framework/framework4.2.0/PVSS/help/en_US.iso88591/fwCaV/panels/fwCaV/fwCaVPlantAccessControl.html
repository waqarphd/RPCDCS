Panel Name

fwCaVPlantAccessControl.pnl
Introduction
 Panel for configuring the Access Control
Instructions

This panels allows to configure the Access Control for the plant, to specify which access rights the user needs to be an Operator or an Expert.

It is possible to have the same configuration for all plant children checking the last check box.
Restrictions If a loop or an area is selected in the Editor & Navigator and the check box for applying recursively the configuration is checked, this configuration will be applied only to the plant children, in this case the plant is not configured.

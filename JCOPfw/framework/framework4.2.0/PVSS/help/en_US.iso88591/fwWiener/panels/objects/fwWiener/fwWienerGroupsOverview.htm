Panel Name

fwWienerGroupsOverview.pnl
Introduction

This panel is intended to be used as a reference panel for showing an overview of the channel groups of a Wiener Marathon.
Instructions

When the panel loads, it displays the channel group information relating to the Wiener device specified $sDpName.

Controls are available to send the switch ON or OFF commands.

The "Manage Groups..." button provides acces to a panel to create more Wiener Marathon channel groups.
Restrictions

None

 

 

Dollar Parameters
Name Description  
$sDpName

The name of the Wiener data point to act on. Must be of type FwWienerPS, FwWienerFanTray or FwWienerCrate

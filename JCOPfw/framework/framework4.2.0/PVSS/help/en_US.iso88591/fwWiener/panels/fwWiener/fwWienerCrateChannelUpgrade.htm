Panel Name

fwWienerCrateChannelUpgrade.pnl
Introduction

This panel is used to upgrade all channels of a Wiener VME crate to the latest models.

This upgrade is applied to change channels from the "Wiener Channel" model to the "Wiener Crate Channel" model. Previously all settings for the crate channels were part of the standard Wiener OPC output group ("WienerOPCGroupOut") which meant that it was always possible to send new setting values to the hardware. With Wiener VME Crate channels this is not desired. By upgrading to the new model and applying the new addressing settings, the channel settings are moved to a new OPC group ("WienerOPCGroupCrateSettingsOut") which is inactive by default.

Once configured in the new way, to apply VME crate channel settings to the hardware, it is necessary to temporarily activate the "WienerOPCGroupCrateSettingsOut" group.

Instructions

When the panel loads, it checks the models of all the VME crate channels. If all have the latest model, then the panel reports this and all actions are disabled. If some channels are of older models, then the panel reports this and enables the actions needed to upgrade the channels.

To upgrade the models of the channels, it is necessary to press the "Upgrade Channels" button. If you wish to reapply the new default addressing, make sure that the "Apply new addressing" check box is checked (this is the default and is strongly recommended). If you do not wish to immediately change the addressing, then you can uncheck this box before pressing the "Upgrade Channels" button.

Note: If you choose not to apply the new addressing, the model of the channels will be changed but the OPC groupings will not be modified, so the deactivation of the settings will not be applied. To do this manually at a later time you can use the "Simple Settings" panel of the Device Editor & Navigator to reapply the new default settings (including the new OPC groupings).

Restrictions

None

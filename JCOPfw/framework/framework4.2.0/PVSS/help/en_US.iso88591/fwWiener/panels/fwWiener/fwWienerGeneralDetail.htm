Panel Name

fwWienerGeneralDetail.pnl
Introduction

This panel is used to show the detailed information about general features of Wiener devices, such as software versions and running times.
Instructions

The panel shows a list of information about the relevant device. Depending on the device type, different proparties will be shown.
Restrictions

None

 

 

Dollar Parameters
Name Description  
$sDpName

The name of the data point to show details of. This data point must be of type FwWienerCrate, FwWienerPS or FwWienerFanTray.

Panel Name

fwWienerManageDrivers.pnl
Introduction

This panel is used to create, delete and configure devices that are used to represent the OPC servers that connect to Wiener hardware.

The devices are not essential for the communication with the hardware, but by creating and configuring the OPC addressing, you can access OPC items at the server level in the Wiener OPC servers.

For Wiener Marathon OPC servers, this enables the viewing of the current up-time of the server. For Wiener KVASER OPC servers, you can see the time of the last driver error and get access to a series of cache words in the OPC server.
Instructions

The panel is divided into three parts - Creating, Configuring addressing and Deleting.

To create a new server device, choose the driver number with which you connect to the server. Then choose if this is a Wiener KVASER OPC server or not. If you want the connection to the server OPC items, you should make sure that the "Default Addressing" check box is checked. When this is done, you press "Create" to create the actual device in PVSS.

If you need to (re-)configure the addressing of the server devices, you can use the middle section of the panel. From here you can choose the driver to operate on by selecting its number. Then you can choose to either "Set Defaults" or "No Hardware Connection". Pressing "Configure" will then apply these settings to the device. Alternatively, you can press the "Modify..." button and configure the addressing by hand.

To delete a server device, use the bottom section of the panel. Choose the driver number that corresponds to the server device you want to delete, then press "Delete".

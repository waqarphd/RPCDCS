Instructions

The table contains the detailed information about the structure of the selected subscription in the Device Editor & Navigator:- which dpes are subscribe under which DIP tags.

  • Data Point Element: DPe subscribed and where the value is stored.
  • DIP Tag: names of the subscription fields.
  • Type: name of the datatype.
  • Value: current value of the subscription fields..

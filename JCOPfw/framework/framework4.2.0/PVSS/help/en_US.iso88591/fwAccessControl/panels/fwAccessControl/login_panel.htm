Introduction

Login panel

The standard PVSS login panel, customized for use with the JCOP Framework.

Figure 1: Login panel
Instructions
To log in:
  • Type in your login name in the "User Name" field and press ENTER
  • Type in your password in the "Password" field and press ENTER

If the username or password is incorrect, you will be informed by an error message.

If you forgot your password or cannot log in, please contact your user administrator.

Note! If your password is empty you will be asked to change your password upon login.

To cancel the login process:
Press the Cancel button
To change the logo (pitcture):
Copy the file (in the BMP format) containing the logo to pictures/logo.bmp . This will alter the logo used by the login panel and the Change Password panel.
Restrictions
  • If the login is unsuccesfull, the previous user is not logged-out automatically. One needs to use the Log Out option available in the context menu brought by the button (embedded on many panels) to perform a log out from the UI.

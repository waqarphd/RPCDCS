Panel Name

fwHVAction.pnl
Introduction
This panel is used to set the value of one of the high voltage parameters of all the devices in a hierarchy below the device passed as parameter (including this device). Before setting the value, it is checked whether the device has the required parameter. Only if the parameter is present the value will be set.
Instructions
The user can press on of the available buttons to set the a high voltage parameter. Currently the following parameters are available:

  • Switch channels ON/OFF: switches ON or OFF all the channels below in the hierarchy from the device where the panel was open.
  • Set v0, v1, i0, i1, Ramp Up Speed, Ramp Down Speed, Trip Time: set the corresponding high voltage parameter in all the channels below in the hierarchy from the device where the panel was open.
As an example, let's imagine we have a crate, with two boards, each with 16 channels. In the panel is open at the level of the crate and v0 is set, it will set it for all the 32 channels, and will do nothing for the boards or the crate. If the panel is open at the level of one of the board, it will set v0 for the 16 channels of that board, and will do nothing for the 16 channels of the other board, the other board itself or the crate.

Panel Name

fwDeviceConversionConfig.pnl
Introduction
 This panel is used to configure the conversion of data when it is sent by the PVSS system (command conversion) or received by it (message conversion). It can be applied to the elements of the Framework device that is passed as parameter. A device definition is required for the device type/model. The panel allows to make settings for individual elements or for a group of them.
Instructions When the panel is opened a table is displayed with all the elements of the device that can have a conversion. The 'Config' column indicates whether the element has already a conversion config (Y) or not (N).  The 'Type' column  indicates the type of conversion that can be applied to the element: 'msg_conv' for message conversion and 'cmd_cnv' for command conversion.

There are three possibilities to change the settings:
  • Individual element
    • By clicking on the name of the element in the table, a pop-up panel will allow you to change the configuration for the element.
  • Selected elements
    • First select the elements on the table. To do this you can use the key combination 'CTRL + click' to select several elements or 'CTRL + SHIFT + click' to select a range in the table. The selected elements have to have the same type of conversion (as listed in the 'Type' column).
    • Click on the 'Configure selected' button. The same panel of the 'Individual element' section will appear, but this time the settings will apply to the selected elements.
  • All elements
    • Click on the 'Configure all' button. The same panel of the 'Individual element' section will appear, but this time the settings will apply to the all the elements in the table. This option will only work if the conversion for all elements is of the same type.

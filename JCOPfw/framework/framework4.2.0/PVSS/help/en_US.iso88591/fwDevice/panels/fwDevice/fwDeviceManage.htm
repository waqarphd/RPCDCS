Panel Name

fwDeviceManage.pnl
Introduction
 This panel gives an overview of the children of the Framework device passed as parameter and also allows to do some settings for the device itself and its children.
Instructions When the panel is opened a table is displayed with all the children devices of the device passed as parameter in the current hierarchy. If the device has slots where the children are included, the table will show them.

The following possibilities are offered to the user when the panel is opened in Editor mode in the Hardware view:
  • Add new children: this can be done either by clicking on a free slot or pressing the 'Add' button.
  • Remove children: select the devices you want to delete in the table and then press the 'Remove' button. This will remove from the system the selected devices and its children
  • Set up children: select the children devices you want to set up and then press 'Settings'. This will pop up the Device Simple Configuration panel, where you can choose how to configure the devices.
  • Advanced options: this will pop up a panel made by the device developer to have detailed control of the configuration of the current device. This action is meant only for experts. This button is optional, it is only available if the device developer considered it necessary.

When the panel is open in Navigator mode or in the Logical view, the actions mentioned above are not available. In this case the panel simply lists the children of the current device.

Panel Name

fwDeviceLogicalSelectAliases.pnl
Introduction
 This panel is used to select the device aliases that will be used when adding devices from the Hardware view to the Logical view.
Instructions
When the panel is open, the table lists the devices available to be included in the Logical tree. The column 'Device Alias' proposes a default alias that will be used in the Logical tree (the device alias will be appended to the logical parent device dp alias to have the device dp alias).

There are two ways to change the default alias. One is to click on a cell and type the new alias. The other way is to use the controls in  'Choose alias for devices'. There you can specify how to change the default alias for all the devices in the table or only the selected ones. As an example, imagine we want our devices to be called wire001, wire002, etc in the Logical view (see Figure 1). Then we would choose 'Alias root' = wire, 'Number of digits' = 3 and 'Starting number' = 1. After we would press either 'Update all' or  'Update selected' and the 'Device Alias' column would be updated. 

Finally we have to add the devices to the Logical tree by pressing 'Add all' or 'Add selected'.

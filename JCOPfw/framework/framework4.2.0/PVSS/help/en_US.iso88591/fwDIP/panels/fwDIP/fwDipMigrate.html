Panel Name

fwDipMigrate.pnl
Introduction
 This panel is used to migrate the data stored in previous version of DIP DPT into the new structure.
Instructions The left side of the panel contains the old version DPs that have any data in them. The right side has a list of new version of DIP config DPs. Clicking the "Migrate" button moves the data from the selected old config DPs into the new one.
Restrictions The panel eliminates the duplicates - so clicking the button multiple times has no negative effect.

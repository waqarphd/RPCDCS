Panel Name

fwDeviceSimpleConfiguration.pnl
Introduction
 This panel is used to set the default configuration for the elements of a device and all of its children in the hierarchy. The defaults for hardware connection, alarms and archiving are entered by the device developer in the device definition.
Instructions
The usage of this panel is very simple:

1. Select in the comboboxes the action you want to apply to each of the 3 possible configurations (Hardware connection, Archive, Alarms). The available actions are:
    • Don't modify: leaves the configuration as it is
    • No Hardware Connection or No Archive or No Alarms: removes the configuration if it exists
    • Set Defaults: configures with the default settings specified by the device developer
2. Press the 'Ok' or 'Apply' buttons. This will perform the selected configurations on the current device and all of its children.

The 'Advanced' button will take you to a panel meant for experts where one can modify individually any of of the settings. This button is optional, it is only available if the device developer considered it necessary.

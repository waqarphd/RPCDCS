Component Name
 fwCoolingAndVentilation
Description

This component allows to parser a SCY file to create the areas and loops for a Cooling and Ventilation system.

It makes possible to configure these devices and manage Synoptics Bitmap instances and widgets.

After installing the component is necessary to have an SCY file to create the structure that this file contains, and then to use the panel fwCaVScyParser.pnl to generate the data in PVSS. The user can get this panel selecting the CaV node in the Editor & Navigator and in the Editor mode clicking over the Advanced options button.

When the file is imported and when the user click over the button Commit changes, all data are generated and the Poll Group and the ModBus Driver are configured. The Poll Group is configured with a Poll time by default (10000 ms), this value can be changed manually in the System Management.

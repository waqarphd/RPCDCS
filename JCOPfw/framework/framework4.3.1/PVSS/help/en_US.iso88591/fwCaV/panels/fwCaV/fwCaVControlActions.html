Panel Name

fwCaVControlActions.pnl
Introduction
 Panel for advanced setting of the cooling plant/loop
Instructions

This panel allows to manipulate with the control word. Additionally it displays the information about current control word and the readback values.

All the changes are applied only when Apply button is pressed.
Restrictions Only a user who is an expert can access to this panel if the Access Control is installed and configured

Dollar Parameters
Name Description  
$sDpName Datapoint of the plant/loop  
$sMyReferenceName Name of the reference panel itself (defined when inserting the reference panel with GeDI, or when adding with addSymbol)

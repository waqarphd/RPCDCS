Panel Name

fwAlarmHandlingScreenOptions.pnl
Introduction

This screen is used to configure the JCOP Framework Alarm Screen.
Instructions

Alarm Table Options

Show Device DP Name: Show/hide the column "DP Name"

Show Logical Name: Show/hide the column "Logical Name"

Show Description: Show/hide the column "Description" with visibility options.

Show timestamp: If an alarm is in WENT status, you can choose wether to show the time for CAME or the WENT event.

Show First Alarm Value: Show/hide the column "Value" (that shows the value that triggered the alarm).

Show Online Value: Show/hide the column "Online" (that shows the current value, with customizable refresh rate).

Use alarm colour...: The full line will be coloured with the alarm colour.

Lines for Historical Alarms: When in Hisorical Mode, this is the maximum amount of alarms to be shown.

Screen Options

Display status of currently displayed remote systems: Show/hide the box with connected systems counter.

Keep scroll position for at least: Indicate how many seconds you want the Screen to stay in the currently selected alarm, before going back down to the line of the latest occurred alarm.

Show default filter when idle for: Reset the filter to the default one, if the operator did not intercat with the Alarm Screen.

Default filter on startup/idle: Filter to be loaded at startup, or to be used after idle interval timeout.

Show all alarms: Show all the alarms, single and summary.

Use alarm reduction: Show only the summary alarm if the threshold of single alarms is reached. See the PVSS help for more details on Alert Reduction.

Hide all summary alarms: Show only the single alarms.

Alarm Help formats...: Open the setup panel for help pages to be opened as alarm help.

Access Control...: Open the setup panel to protect the command buttons on the Alarm Screen.
Restrictions

None

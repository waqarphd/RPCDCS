Panel Name

fwWienerMarathonOperation.pnl
Introduction

This panel is used to monitor and operate the Wiener Marathon power supply framework devices.
Instructions

The operation panel is divided into several sections. Along the top is the general information like data point name, description and the IP number of this Marathon device. There is also the possibility to go to the framework "Action" panel to peform actions on alerts and an indication of the state (using the framework colour convention) of the summary alert of the device.

The left hand side of the panel gives information on the current readings from the hardware. The current Power state is shown along with an indication if all the channels are OK and the current up-time of the device.

At the bottom left, there are buttons to switch the power supply ON or OFF.

The right hand side of the panel gives an overview of the channels inside the device. There are two possible views:

1. Viewing all the channels in the device - see Fig 1.
2. Viewing the channel groups that have been defined - see Fig 2. For each group you can view the channels in the group and switch the whole group of channels ON or OFF. In addition, the "Manage Groups..." button allows the creation ,deletion and configuration of Groups. These Groups must be created in PVSS in order to allow the control of the channel groups in the Wiener Marathon device.
Restrictions

None

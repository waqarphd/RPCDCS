Component Name
 fwMCS
Description

Provides communication between PVSS and DIP.

Selecting an experiment this component provides the possibility of adding / deleting subscriptions for the magnet publications of this experiment.
Dependencies
 fwCore, fwDIP
Additional software required
 None


Panels
This panel is used to subscribe and unsubscribe DIP publications for the selected experiment.
 fwMagnetSubscription.pnl
This panel is used to display the details about the selected subscription.
 fwMagnetOperation.pnl

Panel Name

fwTrendingTimeSpec.pnl
Introduction

This panel is used to select a length of time to be used as the time range for a plot display.
Instructions

When the panel loads, the current setting of the time range is displayed.

You can adjust the time range with the spin buttons to the required value.

Press OK to confirm the new time range or press cancel to close the panel without modifying the settings.
Restrictions None

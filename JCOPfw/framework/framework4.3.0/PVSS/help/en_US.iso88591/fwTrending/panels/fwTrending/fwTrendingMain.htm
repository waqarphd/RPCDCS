Panel Name

fwTrendingMain.pnl
Introduction

This panel is used to build, maintain and browse a tree structure of Trending Plot and Pages.
Instructions

This panel runs in two modes in the same way as the Device Editor and Navigator. There is "Editor" mode for constructing the tree structure and configuring the plot and page items. The "Navigator" mode is for browsing the tree and viewing the plot and page displays. It is possible to switch between modes by pressing the "Go to Editor" or "Go to Navigator" button.

The tree works in the same way as the Device Editor and Navigator. Select and double click a row to expand or contract nodes that have children. By selecting a row and right clicking a menu of options appears with a list of actions that can be taken. The actions that are available depend both on the mode (Editor or Navigator) as well as the currently selected item in the tree. When selecting an item in the tree, the device type and data point name are shown just below the tree.

Here is a brief overview of the actions available in Editor mode:

Add... - to add a node, new plot/page or existing plot/page below the currently selected device
Remove - to remove either the selected item, the item and all its children or only the children of the node
Cut - cut the current item (and subtree) and place it in the clipboard node of the tree
Paste - paste the contents of the clipboard below the selected item
Rename - to rename the current item in the tree
Reorder - to reorder the order in which the children of the current item are shown in the tree
Settings - configure the selected Trending Page or Trending Plot
Template Parameters - set the values for any template parameters required by the Trending Page or Trending Plot

Here is a brief overview of the actions available in Navigator mode:

View - opens a display of the selected Trending Page or Trending Plot

In addition, pressing the "Manage Plots/Pages" button takes you to a screen where you can browse all the trending pages and plots that exist in your system and any other remote systems that are connected, whether or not they present in any Trend Tree.

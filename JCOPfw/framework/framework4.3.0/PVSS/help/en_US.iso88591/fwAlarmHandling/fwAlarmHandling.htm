Component Name
 fwAlarmHandling
Description

Allows for the filtering and display of alarms in a PVSS system. A JCOP Framework specific Alarm Screen is provided which gives enhanced functionality over the standard PVSS alarm screen.
Dependencies
 None - distributed as part of the fwCore.
Additional software required
 None


Panels
The JCOP Framework Alarm Screen to view and filter alarms in your system
 fwAlarmHandlingScreen.pnl
Alarm Screen Options
 fwAlarmHandlingScreenOptions.pnl
The JCOP Framework Alarm Groups Screen to view alarms in your systems and group them following your criteria
 fwAlarmHandlingGroupsScreen.pnl
Alarm Groups configuration
 fwAlarmHandlingGroupsConfiguration.pnl

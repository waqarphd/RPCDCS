Panel Name

fwCaVLoopOperation.pnl
Introduction
 Panel for operating the cooling loop
Instructions

This panel provides the information about the cooling loop operation and allows to issue various loop related actions. It is divided into several sections (composed of separate reference panels):

  • Access Control - If the Access Control is installed and configured and depending on user privileges, the user could be:
    • An Expert: will be able to initiate any action.
    • An Operator: will be able to see everything and will be able to initiate any action except setting a new value for a parameter or acknowledging alerts.
    • A Guest: will be able to see everything but not initiate any action.
  • Loop Operation - composed of yet few more reference panels:
    • Loop Status information combined with basic loop operation. It shows the information extracted from loop status word and allows to modify the basic loop state. For more advanced settings one should go to Actions.
    • Commands - Actions invoke advanced editor of loop state
  • Loop Bitmap - if there is any Synoptics Bitmap associated with the loop, then the button allowing to view the Bitmap is active.
  • Alarms / Warnings information - contains the LED indicator of the alarms and warnings that are defined for the given loop type.
  • Global Timeout - shows the time till next changes are allowed

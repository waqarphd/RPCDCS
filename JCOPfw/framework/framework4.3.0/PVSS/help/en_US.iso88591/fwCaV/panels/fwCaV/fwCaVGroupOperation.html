Panel Name

fwCaVGroupOperation.pnl
Introduction
 Panel for operating the cooling group (area)
Instructions

This panel provides the information about the cooling group (area, group of loops, etc.). It allows to issue various group related actions. It is divided into several sections (composed of separate reference panels):

  • Access Control - If the Access Control is installed and configured and depending on user privileges, the user could be:
    • An Expert: will be able to initiate any action.
    • An Operator: will be able to see everything and will be able to initiate any action except setting a new value for a parameter or acknowledging alerts.
    • A Guest: will be able to see everything but not initiate any action.
  • Group Bitmap - if there is any Synoptics Bitmap associated with the group, then the button allowing to view the Bitmap is active.
  • Group parameters button - to view/alter group parameters
  • Children overview - list and basic information about underlying loops. It is possible to modify at the same time the basic state for the selected loops or for all loops in the table.
  • Global Timeout - shows the time till next changes are allowed

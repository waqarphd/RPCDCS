Introduction
 Panel with overview of alarm (or warning) words
Instructions

Panel displas all the alarm (or warning) words in a tabular form.

Each table represents one word. It contains the alarm/warning description and the bit value (whether given alarm is on or off).

One can click rows in the table to get more info about particular bit

Panel Name

fwCaVPlantOperation.pnl
Introduction
 Panel for operating the cooling plant
Instructions

This panel provides the information about the cooling plant operation and allows to issue various plant related actions. It is divided into several sections (composed of separate reference panels):

  • Access Control - If the Access Control is installed and configured and depending on user privileges, the user could be:
    • An Expert: will be able to initiate any action.
    • An Operator: will be able to see everything and will be able to initiate any action except setting a new value for a parameter or acknowledging alerts.
    • A Guest: will be able to see everything but not initiate any action.
  • Plant Operation - composed of yet few more reference panels:
    • Plant Status information combined with basic plant operation. It shows the information extracted from plant status word and allows to modify the basic plant state. For more advanced settings one should go to Actions.
    • Alarms / Warnings information - contains the LED indicator of the alarm and warning as well as buttons to invoke the more detailed information
    • Commands - Actions invoke advanced editor of plant state
  • Children Overview - contains the list of all plant children. As children we understand all the elements below the plant in the Device Editor Navigator hierarchy - and most probable children are loops. It is possible to modify at the same time the basic state for the selected loops or for all loops in the table.
  • Plant Bitmap - if there is any Synoptics Bitmap associated with the plant, then the button allowing to view the Bitmap is active.

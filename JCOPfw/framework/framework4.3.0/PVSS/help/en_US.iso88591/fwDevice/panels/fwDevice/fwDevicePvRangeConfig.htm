Panel Name

fwDevicePvRangeConfig.pnl
Introduction
 This panel is used to configure the range of values for the elements of the Framework device that is passed as parameter. A device definition is required for the device type/model. The panel allows to make settings for individual elements or for a group of them.
Instructions When the panel is opened a table is displayed with all the elements of the device that can have a range defined. The 'Config' column indicates whether the element has already a range defined (Y) or not (N). 

There are three possibilities to change the settings:
  • Individual element
    • By clicking on the name of the element in the table, a pop-up panel will allow you to change the configuration for the element.
  • Selected elements
    • First select the elements on the table. To do this you can use the key combination 'CTRL + click' to select several elements or 'CTRL + SHIFT + click' to select a range in the table.
    • Click on the 'Configure selected' button. The same panel of the 'Individual element' section will appear, but this time the settings will apply to the selected elements.
  • All elements
    • Click on the 'Configure all' button. The same panel of the 'Individual element' section will appear, but this time the settings will apply to the all the elements in the table.

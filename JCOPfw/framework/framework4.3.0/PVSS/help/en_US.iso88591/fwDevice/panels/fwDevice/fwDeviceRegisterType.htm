Panel Name

fwDeviceRegisterType.pnl
Introduction
 This panel allows to declare an existing PVSS datapoint as Framework Device. It will simply create an empty device definition that can be edited afterwards with the panel fwDeviceEditDefinition. The existence of a definition is required for some of the Framework functionality to work.
Instructions When the panel is opened a table is displayed with all the Framework devices that are available in the system. The first column shows the Framework device type and the second the PVSS datapoint type.

One can register a new type by selecting in the 'Non registered DP types' combox one of the PVSS datapoint and entering a device type for it.  Then pressing 'Register' will create the definition and refresh the table.

Panel Name

fwDeviceAddress.pnl
Introduction
 This panel is used to configure the connection to the hardware for the elements of the Framework device that is passed as parameter. A device definition is required for the device type/model. The panel allows to make settings for individual elements or for a group of them.
Instructions When the panel is opened a table is displayed with all the elements of the device that can be connected to the hardware. The 'Config' column indicates whether the element has already a hardware address defined (Y) or not (N).  The 'Direction' column  indicates whether the element is read from the hardware or it is a command.

There are three possibilities to change the settings:
  • Individual element
    • By clicking on the name of the element in the table, a pop-up panel will allow you to set the address for the element.
  • Selected elements
    • First select the elements on the table. To do this you can use the key combination 'CTRL + click' to select several elements or 'CTRL + SHIFT + click' to select a range in the table.
    • Click on the 'Configure selected' button. A panel with a reduced set of parameters compared to the panel of the 'Individual element' section will appear.  The parameters specific to each element are taken from the internal definition for the device.
  • All elements
    • Click on the 'Configure all' button. A panel with a reduced set of parameters compared to the panel of the 'Individual element' section will appear.  The parameters specific to each element are taken from the internal definition for the device. The settings will apply to the all the elements in the table.
It has to be noted that the 'Configure selected' and 'Configure all' buttons are only available when the device has a default configuration for the addresses.

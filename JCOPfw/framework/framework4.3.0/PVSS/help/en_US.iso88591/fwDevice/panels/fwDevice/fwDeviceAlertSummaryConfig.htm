Panel Name

fwDeviceAlertSummaryConfig.pnl
Introduction
 This panel is used to configure the alarm summary of the Framework device that is passed as parameter. A device definition is required for the device type/model.
Instructions There are two use cases for this panel:
  • Create/Delete summary
    • This can be done with the 'Create' and 'Delete' buttons. If the summary exists, only the 'Delete' button will be available. If the summary doesn't exists, only the 'Create' button will be available.
  • Add/Remove alarms to the summary
    • If the summary exists, the controls to add and remove alarms to it will be enabled.
    • The panel offers the possibility to add to the summary alarms defined for elements of the current device, or existing summary alarms of the children devices.
    • Once you are happy with the included/excluded sets, don't forget to press the 'Apply' button, so that the actual configuration takes place.
Restrictions Currently only works in the Hardware View. It is not possible to add logical children alarm summaries.

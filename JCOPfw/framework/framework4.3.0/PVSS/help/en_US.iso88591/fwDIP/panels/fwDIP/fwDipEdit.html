Panel Name

fwDipEdit.pnl
Introduction
 This multi-function panel allows to manage the DIP config DPs, subscribe and unsubscribe from the DIP publications and to publish and unpublish data from PVSS.
Instructions The header of the panel contains the button which allows the selection of the mode of operation (modes of operation will be explained below) and a selection box for selecting the DIP API manager config DP. Each DIP API manager uses a separate config.

DIP Manager section provides some basic operations on the API Manager and on the DIP config DPs. There are three sections:

  • Config related actions - adding new config and removing the existing one. These options should not be generally used.
  • Browser Cache - allows to issue the command to the API Manager to regenerate the cache with the DIP hierarchy. It is necessary to view newly published publications.
  • Queries - allows to specify how long to wait for the response from the API Manager. The default should not be tweaked.

Subscribtions - The combo box "Subscribed Services" contains the list of all the DIP publications that the API manager is currently subscribed to (it is analogous to list in the fwDipView in "Subscribed Services" view). The table below contains the detailed information about subscriptions of selected DIP publication - which dpes are being subscribed to which DIP tags.
Panel allows to unsubscribe from all the dpes in the table or to unsubscribe only the selected dpe. These operations are invoked by using "Unsubscribe" and "Unsubscribe dpe" buttons

The "New Subscription" part of the panels allows to subscribe to:

  • simple publications (*)
  • single fields of complex publications (*)
  • complex publications (**)
The combo box allows to choose the mode of operation - simple (*) and complex (**). Depending on the selection the displayed fields change.
For simple mode - the user has to select the publication by means of the browser and select the dpe from the selector.
For complex mode - the user has to select the publication and then after clicking the "Add Subscription" button the panel for complex subscribtion will appear (fwDipEditDptCreate).

Publications - The combo box "Published Services" contains the list of all the publications that are being published (it is analogous to list in the fwDipView in "Published Services" view). The table below contains the detailed information about the structure of the selected publication - which dpes are being published under which DIP tags.
In order to unpublish the publication it is necessary to click the "Del Publication" button.
Creation of the new publication is started by clicking the "Add Publication" button. Then one has to insert at least one tag into the publication by filling the fields in the bottom part of the panel. It is necessary to click "Save changes" button in order to finish the creation of the publication.
One can also modify the existing publication - it is possible to add additional fields or remove existing ones (by selecting a row from a table and clicking the "Del dpe" button. All changes are commited only after clicking the "Save changes" button. In order to reset the data (undo the unsaved changes), one has to click "Reload" button

Restrictions The values of the dpes are not updated dynamically.

Panel Name

fwAlarmOperationPopupPanel.pnl
Introduction

This panel is used to view the alert configuration for a data point element. The current alert configuration is shown and the current state of the alarm is indicated by a coloured box.

The user can also mask (deactivate), unmask (activate) and acknowledge alarms as appropriate.
Instructions

When the panel loads, it reads the current alert configuration and displays this, along with the current state of the alert and the name of the data point element which is being viewed.

Three buttons are available to perform actions. These buttons are disabled when the action is not relevant to the current state of the alarm. The buttons are:

Mask - Mask the alarm. The alert config is deactivated. The configuration is kept but no alarm is generated.

Unmask - Unmask the alarm. The alert config is activated.

Acknowledge - Acknowledge the alarm.
Restrictions

This panel can be used to view analog and digital style alarms but no summary alarms. For summary alarms, use fwAlarmSummaryOperationPopupPanel.pnl.

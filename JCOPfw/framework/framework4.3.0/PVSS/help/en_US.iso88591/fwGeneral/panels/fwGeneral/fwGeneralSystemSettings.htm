Panel Name

fwGeneralSystemSettings.pnl
Introduction
 This panel is used to choose the browser to be used to display the Framework online help.
Instructions It is possible to choose a different browser command for Windows and Linux systems. This is to take into account scattered systems.
The default command in windows (cmd /c start "" $1) will start the default web browser of the system.
Restrictions
  • In a scattered system the help will only be available if the remote UI has file system access to the Framework installation.
  • In some systems it may take a while when opening the first help page.

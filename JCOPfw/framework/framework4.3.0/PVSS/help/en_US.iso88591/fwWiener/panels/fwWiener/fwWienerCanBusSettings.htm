Panel Name

fwWienerCanBusSettings.pnl
Introduction

This panel is used to set certain configurable parameters relating to the CAN bus used for Wiener devices. The settings entered here are included in the OPC server XML config file if this is exported using the fwWienerDriverConfiguration panel.
Instructions

When the panel loads, it displays the current values stored in PVSS for the given parameters. These are the CAN bus baud rate as well as the read and write queue lengths.

New settings can be entered as desired. To save the settings, press the Apply button. To close the panel without changing the settings, press the Close button.
Restrictions

The values set in the panel are not automatically applied to the Wiener CAN bus. The panel just saves the settings to PVSS. To actually apply the new settings to the CAN bus, you should reexport the XML config file for the OPC server. This file is generated from the fwWienerDriverConfiguration panel.

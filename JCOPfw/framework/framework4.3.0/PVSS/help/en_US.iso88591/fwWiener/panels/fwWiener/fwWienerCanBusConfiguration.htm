Panel Name

fwWienerCanBusConfiguration.pnl
Introduction

This panel is used to set the configuration of the Wiener CAN bus framework devices.
Instructions

When the panel loads, it displays the current configuration of the Wiener CAN bus device. The device data point name and CAN bus ID are also shown.

The description of the device can be modified by entering the new description in the "Description" field and then pressing the enter key to save the new value.

Tabs are provided for switching between various different types of configuration. Tabs are available for setting the addressing, archiving, alarms and a summary alarm for the device. These are the standard framework panels for this type of configuration.

Particular to the Wiener CAN bus is the "CAN Bus Settings" tab. This allows for configuration of certain specific parameters relating to the Wiener CAN bus. For more details see the fwWienerCanBusSettings.
Restrictions

None

 

 

Dollar Parameters
Name Description  
$sDpName

The name of the data point to configure. This data point must be of type FwWienerCanBus.

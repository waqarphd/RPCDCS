Panel Name

fwWienerDriverOperation.pnl
Introduction

This panel is used to monitor Wiener OPC servers used to connect to the Wiener hardware.

For the KVASER based OPC server, access is given to set and monitor the cache words in the Wiener OPC server. This can be useful with debugging to ensure that the server is working correctly, regardless of whether or not any Wiener devices are available to be controlled. For the Marathon OPC server, only the current up-time of the server is given.
Instructions

When the panel loads, it displays the information relating to the default Wiener Marathon server (Driver number 14).

By selecting a different number from the "Driver Number" combo box it is possible to view the information relating to other Wiener OPC servers. The default KVASER OPC server driver number is 11.

When a Marathon server is selected, the current up-time of the server is given - see Fig 1.

When a KVASER server is selected, a description is shown as well as the time stamp of the last driver error - see Fig 2. In addition, there is a table which shows the status of the cache words in the Wiener OPC server. The last set value is shown in the column "Setting" and the readback of the value from the OPC server is shown in the "ReadBack" column. To set a new value for one of the cache words, simply click on the corresponding row of the table in the "Settings" column.

Panel Name

fwWienerDriverDetails.pnl
Introduction

This panel is intended to be used as a reference panel for showing the Wiener Driver cache words.
Instructions

When the panel loads, it displays the information relating to the Wiener driver device specified $sDpName.

The time stamp of the last driver error (if any). In addition, there is a table which shows the status of the cache words in the Wiener OPC server. The last set value is shown in the column "Setting" and the readback of the value from the OPC server is shown in the "ReadBack" column. To set a new value for one of the cache words, simply click on the corresponding row of the table in the "Settings" column.
Restrictions

None

 

 

Dollar Parameters
Name Description  
$sDpName

The name of the FwWienerDriver data point to act on.

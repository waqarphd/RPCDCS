Panel Name

fwWienerMarathonRCMChannelOperation.pnl
Introduction

This panel is used to monitor and operate the Wiener Marathon RCM Channel framework devices.
Instructions

The operation panel is divided into several sections. Along the top is the general information like data point name, description and the position of the channel in its parent device. There is also the possibility to go to the framework "Action" panel to peform actions on alerts and an indication of the state (using the framework colour convention) of the summary alert of the device.

The left hand side of the panel gives information on the current readings from the hardware. The current Power state is shown along with status flags for various error conditions. Below this, readouts of voltage, current and temperature are given.

At the bottom left, there are buttons to switch the channel ON or OFF.

The right hand side of the panel gives access to change to settings of this channel. For example, you can change the maximum current and voltage limits, change the behaviour of device when a limit is exceeded or allocating a group number to this channel.
Restrictions

None

 

 

Dollar Parameters
Name Description  
$sDpName

The name of the data point to configure. This data point must be of type FwWienerMarathonChannel with model type "RCM Channel (TCP/IP)".

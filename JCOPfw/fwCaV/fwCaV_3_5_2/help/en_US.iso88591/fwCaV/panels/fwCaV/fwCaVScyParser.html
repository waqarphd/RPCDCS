Panel Name

fwCaVScyParser.pnl
Introduction
 This is one of the most important panels of the fwCaV component. It works with the SCY files (provided by TS/CV) and generates the necessary metadata and data in the PVSS.
Instructions
  • Step 1: Provide the path to the SCY file that is going to be parsed and enter the System Prefix (each system has its own prefix defined in the naming convention - e.g. Tile, Lar). Each of the variables in the SCY file is prefixed with this prefix.
    If the model definition (metadata) for this particular system already exists then you will receive information that update mode is active.
    After providing that information one can proceed with the reading of the SCY file. In case of any errors proper notification should be displayed. If the process terminated without errors then the table in the middle section of the panel should become populated with some data.
  • Step 2: The middle section of the panel is called "Model Definition Overview". It allows to view the variables that had been read in from the SCY file. If the update mode is active it will also show how the data from the SCY file is related to the existing model.
  • Step 3: It is necessary to provide some data about the system that is not contained within the SCY file. The bottom part of the panel is devoted to this action. One need to provide data about the status/control bits that are used by the plant and the loops as well as the descriptions for the loop warnings and alarms. All that data should be contained in the Technical Documentation that is provided by TS/CV.
  • Step 4: Commiting the changes. After pressing the button you will be presented with a panel in which you can specify the IP address for the PLC. The Poll Group and the ModBus Driver will be configured automatically.

NOTE: A button will appear at the bottom of the panel in case that file SCY is a file with references to the areas. This button allows to add the areas to the system. See Fig2.

 
Restrictions The SCY files must be well-formed - according to the naming convention.

Panel Name

fwCaVTrendingPage.pnl
Introduction
 This panel allows to configure a Trending Page.
Instructions

    It is possible to create till 6 plots with 8 curves each one.

  • List on the left contains all the plot instances for the trending page, while the right part of the screen contains all the curves about selected plot instance.
  • Plot instances can be added/removed using the buttons.
  • Curves of a plot can be added/removed/modified. It is necessary to select a line in the table. If this line contains values it is possible to modify or delete its content. If the line is empty it is possible to asign the information to it for seeing the curve in the plot.
  • If the plot will be visible in the page, the Visible option must be marked.
  • Changes to the plot instance are saved using the Save Plot button

Panel Name

fwTrendingManageChildren.pnl
Introduction

This panel is used to manage all the Trending Plots and Pages in your local system as well as any other remote systems which are connected.
Instructions

When the panel loads is shows a list of all the currently existing Plots and Pages in your local system as well as any connected remote systems. For each item, the data point name, title and type are shown.

By double clicking on a row in the table, the configuration panel is loaded for the corresponding plot or page.
By selecting a row then right clicking in the table, the display (operation panel) is shown for the corresponding plot or page.

There is a filtering mechanism available to filter the items in the table (which can be very numerous). It is possible to filter on the data point name, title and type. Data point name and title filters support the * wild card character. To search across many systems you need to include *: at the beginning of your filter. Filtering on type is available to see only Trending Plots, Trending Pages or All (*). Once you have entered the desired filter, press the "Apply Filter" button. To clear any filter and see all Plots and Pages in the list, press the "Clear Filter" button.

In addition, this panel also allows you to create and delete Plots and Pages. To create a plot or page, go to the top right of the panel, select the desired new device type from the combo box and then press "Create".

To delete an existing plot or page, select the item in the table and press the "Delete Selection" button down in the bottom left of the panel.

Finally, the "Refresh List" button can be used to refresh the list of plots and pages. This can be useful if a remote system has been connected or disconnected while the panel was running. The "Refresh List" button redoes the search for plots and pages using the current filter.
Restrictions None

Panel Name

fwTrendingChoose.pnl
Introduction

This panel is used to view and select from a list of all the Trending Plots in your local system as well as any other remote systems which are connected.
Instructions

When the panel loads is shows a list of all the currently existing Plots in your local system as well as any connected remote systems. For each item, the data point name and title are shown.

There is a filtering mechanism available to filter the items in the table (which can be very numerous). It is possible to filter on the data point name and title. Data point name and title filters support the * wild card character. To search across many systems you need to include *: at the beginning of your filter. Once you have entered the desired filter, press the "Apply Filter" button. To clear any filter and see all Plots in the list, press the "Clear Filter" button.

The "Refresh List" button can be used to refresh the list of plots and pages. This can be useful if a remote system has been connected or disconnected while the panel was running. The "Refresh List" button redoes the search for plots using the current filter.

To select a Plot, find it in the list and select the row in the table by clicking on it. Then press "OK" to choose this plot. If you wish to close the panel without changing your choice of plot, press the "Cancel" button. If you wish to explicitly choose no plot, then press the "None" button.
Restrictions None

 

 

Dollar Parameters
Name Description  
$sCurrentChoice The data point name of the currently selected plot (to be shown as highlighted in the list when the panel loads). If no current selection exists, then set to "". Required
$sDpType The data point type of devices to show. In principle this panel could work with devices other than plots but basically this value should always be set to "FwTrendingPlot". Required

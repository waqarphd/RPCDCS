Panel Name

fwTrendingPage.pnl
Introduction

This panel is used to display a Trending Page. The plot configuration is read from a predefined Trending Page data point.

When reading from a Trending Page data point, the page display will respond dynamically to any changes in the configuration that is stored in the Page data point.
Instructions

When the page is displayed, there are two buttons in the top right of the panel which allow for actions on the page. These are:

Page plots - This button provides access to all features that relate to the page. From here you can do many actions on all plots in the page simulataneously, such as setting the time range, freezing all plots, etc. You can also save all the settings of each plot back to there respective plot configuration data points.

At a page level you can print the page and export all visible data to a CSV file. Also, you can modify the page settings with the Page Configuration panel, save settings back to the original page configuration data point and save the current page settings to a new (or existing) page data point.

Close - Closes the panel.

All the plots displayed are scaled version of the Plot Display.
Restrictions None

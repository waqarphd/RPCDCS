Component Name fwAccessControl
Description

JCOP Framework Access Control component

This component provides the mechanism for implementing UI-level access control in user panels. A set of access control administration and management panels and a library of functions is provided as well.

Dependencies
  • fwCore (optional, recommended): the Access Control component may also work without the fwCore installed.
  • Integrated Access Control mode requires that all systems in the distributed system have fwAccessControl version 2.5.0 or higher
Additional software required

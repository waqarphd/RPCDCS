Panel Name

fwAccessControl/fwAccessControl_EditUser.pnl

Introduction

This panel is used to display and edit the details concerning the user.

Figure 1: Edit User Panel in edit mode.

The four text fields define user short name, user full name, user comment and user password. It is the user short name that identfies the user.
The checkbox "Enabled" allows to enable and disable the account.
The non-editable text-field "id:" in the top-right corner presents the numerical, unique user index, as used internally by PVSS.
The list presents the names of groups to which the user belongs.

The same panel is also used when a new user is created; in this case the Show Current Privileges and the Review buttons are not available, and the OK button is replaced by Create button:

Figure 2: Edit User Panel in create mode.

If current user has no privileges to edit user details, the panel is open in "view mode", where editing operations are disabled:

Figure 3: Edit User Panel in view mode.
Instructions
To show the privileges granted to the user:
Click on the Show Current Privileges button; this will bring the User Privileges List panel. Note that this button is not available in "create" mode.
To reset the changes made in this panel:
Click on Reset button; The fields in the panel will reflect again the current settings for the user.
To create a new user:
In the panel running in "create" mode, input at least a valid name for a new user, then click the Create button. Remember to enable the account by setting the Enabled checkbox.
To rename the user:
Edit the data in the "Name" text field, then click the OK button.
To change the full name, comment or password of the user:
Edit the data in corresponding text fields, then click the OK button.
To rename the user:
Edit the data in the "Name" text field, then click the OK button.
To enable/disable user account:
Use the Enabled checkbox.
To edit the list of groups to which the user belongs:
Click the Edit... button next to the list of groups; this will bring the Edit User's Groups panel.
To review the changes made to the list of groups:
Click the Review button next to the list of groups; this will bring the Review Changes panel in a mode displaying the changes in user's group belonings.
To accept the changes:
Click the OK button; this will bring the Confirm User Changes panel asking for confirmation of changes.

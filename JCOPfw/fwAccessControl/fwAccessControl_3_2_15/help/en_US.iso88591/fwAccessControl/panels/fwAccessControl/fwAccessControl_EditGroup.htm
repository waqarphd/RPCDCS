Panel Name

fwAccessControl/fwAccessControl_EditGroup.pnl

Introduction

This panel is used to display and edit the details concerning the group.

Figure 1: Edit Group Panel in edit mode.

The three text fields define group short name, group full name and group comment. It is the group short name that identfies the group.
The non-editable text-field "id:" in the top-right corner presents the numerical, unique group index, as used internally by PVSS
The list presents the names of privileges granted to this group.

The same panel is also used when a new group is created; in this case the Show Users and the Review buttons are not available, and the OK button is replaced by Create button:

Figure 2: Edit Group Panel in create mode.

If current user has no privileges to edit group details, the panel is open in "view mode", where editing operations are disabled:

Figure 3: Edit Group Panel in view mode.
Instructions
To show the users belonging to this group:
Click on the Show Users button; this will bring the Users In Group List panel. Note that this button is not available in "create" mode.
To reset the changes made in this panel:
Click on Reset button; The fields in the panel will reflect again the current settings for the group.
To create a new group:
In the panel running in "create" mode, input at least a valid name for a new group, then click the Create button.
To change the full name or comment of the group:
Edit the data in corresponding text fields, then click the OK button.
To rename the group:
Edit the data in the "Name" text field, then click the OK button.
To edit the list of privileges granted to the group:
Click the Edit... button next to the list of privileges; this will bring the Edit Group Privileges panel.
To review the changes made to the list of privileges:
Click the Review button next to the list of privileges; this will bring the Review Changes panel in a mode displaying the changes in group privileges.
To accept the changes:
Click the OK button; this will bring the Confirm Group Changes panel asking for confirmation of changes.

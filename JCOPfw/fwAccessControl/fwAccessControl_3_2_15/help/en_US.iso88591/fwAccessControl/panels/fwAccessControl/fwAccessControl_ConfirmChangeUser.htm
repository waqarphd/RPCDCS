Panel Name

fwAccessControl_ConfirmChangeUser.pnl
Introduction

This panel is used to ask for confirmation before changes to user settings are applied.

Figure 1: Confirm User Changes panel.
Instructions
To accept the changes:
Click on the OK button.
To go back to Edit User panel:
Click on the Cancel button.
To view the list of changes made to the user:
Click on the Review Changes button; this will bring the Review Changes panel in a mode displaying all changes done to the user.
To cancel the changes:
Click on the Cancel button to go back to Edit User panel, use the Close button there.
Restrictions

Dollar Parameters
Name Description  
$userName Specifies the user name to be displayed. required
$changes Specifies the text with details of the changes; new lines are separated by "\n" characted. required
$addedItems Specifies the dyn_string list of groups, to which the user has have been added. required
$removedItems Specifies the dyn_string list of groups from which the user has been removed. required

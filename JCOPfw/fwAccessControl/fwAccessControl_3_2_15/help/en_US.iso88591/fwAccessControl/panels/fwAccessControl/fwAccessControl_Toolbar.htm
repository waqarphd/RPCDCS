Panel Name

fwAccessControl/fwAccessControl_Toolbar.pnl
Introduction

This is the "toolbar" panel for management of Access Control, i.e. it allows to access the panels that allow to browse/create/delete/modify users, groups and domains.


Access Control Toolbar panel.

The panel contains the following elements (from left to right):

  • the text-field displaying the name of the user that is currently logged-in to this UI
  • the login button
  • the Administer... cascade button
  • the Account... cascade button
  • the standard framework on-line help button
Instructions
To logout from this UI:
Select "Logout" from the "Account..." cascade button
To login as a different user:
Press the "login" button or select Login from the Account... cascade button.
The current user will be logged out and the Login panel will be displayed.
To change the password:
Select Change password from the Account... cascade button.
Standard PVSS Change Password panel will be displayed
To find a user with specific privileges:
Select Find User from the Account... cascade button;
the Privilege holders list panel will be opened.
To manage access control configuration
(browse/modify/create/delete domains groups and users)
use the Administer... cascade button and select:
  • Users: to bring up the User List panel and configure users
  • Groups" to bring up the Group List panel and configure groups
  • Domains" to bring up the Domain List panel and configure domains

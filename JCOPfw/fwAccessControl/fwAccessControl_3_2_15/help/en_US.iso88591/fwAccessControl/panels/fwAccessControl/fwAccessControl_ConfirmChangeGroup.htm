Panel Name

fwAccessControl_ConfirmChangeGroup.pnl
Introduction

This panel is used to ask for confirmation before changes to group settings are applied.

Figure 1: Confirm Group Changes panel.
Instructions
To accept the changes:
Click on the OK button.
To go back to Edit Group panel:
Click on the Cancel button.
To view the list of changes made to the group:
Click on the Review Changes button; this will bring the Review Changes panel in a mode displaying all changes done to the group.
To cancel the changes:
Click on the Cancel button to go back to Edit Group panel, use the Close button there.
Restrictions

Dollar Parameters
Name Description  
$groupName Specifies the group name to be displayed. required
$changes Specifies the text with details of the changes; new lines are separated by "\n" characted. required
$addedItems Specifies the dyn_string list of privileges that have been added. required
$removedItems Specifies the dyn_string list of privileges that have been revoked. required

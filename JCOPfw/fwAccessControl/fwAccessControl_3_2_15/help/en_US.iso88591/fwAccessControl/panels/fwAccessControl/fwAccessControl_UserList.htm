Panel Name

fwAccessControl/fwAccessControl_UserList.pnl
Introduction

This panel presents the list of users and allows to configure them.

User List Panel in edit mode.

If the current user has no System:Administer privilege, the panel will be opened in "view" mode, without possibility of modifications.

User List Panel in view mode.

The panel will automaticaly switch between "edit" and "view" mode, whenever the user privileges changes (e.g. when other user logs in to this UI).

Instructions
To create a new user:
Click on Add button; this will bring the Edit User panel in "create" mode.
To delete a user:
Select the user in the list, then click on Delete button.
To modify or display details of a user:
Select the user in the list, then double-click on it or click the Modify (or View if privileges are not sufficient) button. This will bring the Edit User panel.
Restrictions
  1. In "view" mode, where users has no System:Administer privileges, the users cannot be modified/created/deleted

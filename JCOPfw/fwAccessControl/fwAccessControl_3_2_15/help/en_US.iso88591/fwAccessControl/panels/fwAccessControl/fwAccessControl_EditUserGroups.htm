Panel Name

fwAccessControl_EditUserGroups.pnl
Introduction

This panel is used to configure the groups to which the user belongs.

The list on the left side presents all defined groups. The list on the right side presents the groups to which the user will belong:

Figure 1: Edit Users' Group panel

Both lists support multiple-selection (hold the CTRL key when clicking on the item. The items that are alreade present in the right-side list cannot be selected in the left-side list; they are marked by displaying them on the grey background.

Instructions
To add membership is group(s):
Select the group(s) in the left list then click the   >  . One may also add membership in all groups using the  >>  button. The groups that were added will appear in the righ-side list and will be greyed-out in the left-side list.
To remove membership is group(s):
Select the group(s) in the right list then click the   <  . One may also remove membership in all groups using the  <<  button. The groups that were removed will disappear in the righ-side list and will be un-greyed in the left-side list.
To reset the changes done in this panel:
Click on the Reset button. This will return the panel to the state it had when opened.
To apply the changes and close this panel:
Click on the OK button. Note that the changes will not be applied until you apply all changes to the group in User Edit panel by clicking the OK button there.
To cancel the changes and close this panel:
Click on the Cancel button.

Panel Name

fwAccessControl/fwAccessControl_DomainList.pnl
Introduction

This panel presents the list of domains and allows to configure them.

Domain List Panel in edit mode.

The first domain is a "system domain", i.e. it reflects the PVSS system which is running. The other domains are user-created domains.


If the current user has no System:Administer privilege, the panel will be opened in "view" mode, without possibility of modifications.

Domain List Panel in view mode.

The panel will automaticaly switch between "edit" and "view" mode, whenever the user privileges changes (e.g. when other user logs in to this UI).

Instructions
To create a new domain:
Click on Add button; this will bring the Edit Domain panel in "create" mode.
To delete a domain:
Select the domain in the list, then click on Delete button.
To modify or display details of a domain:
Select the domain in the list, then double-click on it or click the Modify (or View if privileges are not sufficient) button. This will bring the Edit Domain panel.
Restrictions
  1. In "view" mode, where users has no System:Administer privileges, the domains cannot be modified/created/deleted
  2. System domain cannot be deleted
  3. System domain has the same name as the running system
  4. UNICOS-specific restrictions (apply when UNICOS access control scheme is active):
    1. the number of domains is fixed to eight; cannot delete or add domain
    2. system domain has no sence

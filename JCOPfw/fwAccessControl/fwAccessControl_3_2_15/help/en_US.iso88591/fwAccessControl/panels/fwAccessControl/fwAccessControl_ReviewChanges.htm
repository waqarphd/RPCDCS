Panel Name

fwAccessControl_ReviewChanges.pnl
Introduction

This panel is used to present the changes. The panel may be displayed in one of the following contents.

  • group changes:
    In this context, the panel shows all changes of settings for a group, which were performed in the Edit Group panel. It is displayed when Review Changes button in the Confirm Group Changes panel is clicked.
    Figure 1: Review Changes panel showing changes to a group.
  • user changes:
    In this context, the panel shows all changes of settings for a user, which were performed in the Edit User panel. It is displayed when Review Changes button in the Confirm User Changes panel is clicked.
    Figure 2: Review Changes panel showing changes to a user.
  • group privileges:
    In this context, the panel shows only the changes done to group privileges. It is called when the Review button is clicked in the Edit Group panel:
    Figure 3: Review Changes panel showing changes to group privileges.
  • users' groups:
    In this context, the panel shows only the changes done to the list of groups to which the user belongs. It is called when the Review button is clicked in the Edit User panel:
    Figure 4: Review Changes panel showing changes to user's groups.
Instructions
To close the panel:
Click on the Close button.
